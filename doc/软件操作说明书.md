## 一、软件概述

### 1.1 软件名称
WaveCast（声波播客）

### 1.2 版本信息
- 版本号：1.0.3
- 版本代码：1000300
- 发布日期：2025年12月
- 著作权人：陈云亮
- 联系邮箱：676814828@qq.com

### 1.3 软件简介
WaveCast 是一款基于 HarmonyOS 平台开发的专业播客管理应用。该应用采用 ArkTS 开发语言，提供了完整的播客订阅、播放、下载和管理功能，支持 RSS/Atom 格式的播客源。

### 1.4 主要功能
- 播客订阅与搜索（支持 RSS/Atom 格式）
- 单集播放与管理
- 离线下载功能
- 多速播放（0.5x - 3.0x）
- 睡眠定时器
- 播放队列管理
- 播放进度保存与同步
- 后台播放支持
- 主题切换
- 网络设置

### 1.5 系统要求
- 操作系统：HarmonyOS API 12 及以上
- 设备类型：手机、平板、2in1设备
- 最低 API 版本：12
- 目标 API 版本：12

### 1.6 必需权限
- 网络访问权限（ohos.permission.INTERNET）
- 网络状态权限（ohos.permission.GET_NETWORK_INFO）
- 后台运行权限（ohos.permission.KEEP_BACKGROUND_RUNNING）

## 二、安装与启动

### 2.1 安装步骤
1. 下载 WaveCast 安装包（.hap 文件）
2. 在 HarmonyOS 设备上打开安装包
3. 按照系统提示完成安装
4. 首次启动时授予必要的权限

### 2.2 权限授权
应用首次启动时会请求以下权限：
- **网络访问权限**：用于获取播客信息和下载音频文件
- **网络状态权限**：用于检测网络连接状态
- **后台运行权限**：用于支持后台播放音频

建议全部授予，以获得完整功能体验。

### 2.3 应用启动
点击桌面图标启动应用，首次启动会初始化数据库和相关服务。

## 三、主界面导航

### 3.1 界面布局
应用采用底部标签栏设计，包含四个主要标签页：
- **订阅**：显示已订阅的播客列表
- **单集**：显示所有播客的单集列表
- **队列**：显示待听播放队列
- **下载**：管理已下载和正在下载的内容

### 3.2 顶部导航栏
- 左侧：应用名称 "WaveCast"
- 右侧：搜索按钮（🔍）和设置按钮（⚙️）

### 3.3 迷你播放器
当有正在播放或暂停的内容时，底部会显示迷你播放器，包含：
- 封面图片
- 单集标题（跑马灯滚动）
- 播放进度条
- 播放时间显示
- 睡眠定时器倒计时（如已设置）
- 播放/暂停按钮
- 下一首按钮

点击迷你播放器可进入完整播放页面。

## 四、功能详细说明

### 4.1 播客订阅功能

#### 4.1.1 搜索播客
1. 点击顶部导航栏的搜索图标（🔍）
2. 进入搜索页面
3. 输入播客名称或 RSS 地址
4. 点击搜索结果查看详情

#### 4.1.2 订阅播客
1. 在播客详情页点击"订阅"按钮
2. 系统会自动获取播客的所有单集信息
3. 订阅成功后，播客会显示在"订阅"标签页中

#### 4.1.3 取消订阅
1. 进入播客详情页
2. 点击"取消订阅"按钮
3. 确认后，播客及其所有单集将从本地数据库删除

#### 4.1.4 刷新播客
- 新订阅的播客会在第一分钟内每5秒自动刷新一次
- 之后每2小时自动刷新一次
- 可手动在播客详情页下拉刷新

### 4.2 播放功能

#### 4.2.1 播放单集
有以下几种方式播放单集：
1. 在单集列表中点击"▶ 播放"按钮
2. 在单集详情页点击播放按钮
3. 在下载列表中播放已下载的单集
4. 在队列中播放排队的单集

#### 4.2.2 完整播放页面
点击迷你播放器或播放中的单集可进入完整播放页面，包含以下功能：

**基本控制：**
- 播放/暂停按钮（中央大按钮）
- 快退15秒按钮（左侧）
- 快进30秒按钮（右侧）
- 进度条拖动（精确定位）

**播放速度调节：**
支持以下播放速度：
- 0.5x - 慢速学习
- 0.75x - 较慢
- 1.0x - 正常速度
- 1.25x - 稍快
- 1.5x - 较快
- 1.75x - 很快
- 2.0x - 2倍速
- 2.5x - 极快
- 3.0x - 最快

点击播放速度按钮（🚀），从菜单中选择所需速度。**速度设置会被记忆，下次播放时自动应用。**

**睡眠定时器：**
1. 点击定时器按钮（⏰）
2. 选择定时时长：5分钟、15分钟、30分钟、45分钟、60分钟
3. 倒计时结束后自动暂停播放
4. 可随时关闭定时器

**其他功能：**
- 添加到队列：将当前单集加入待听队列
- 更多选项（⋮）：
  - 查看单集详情
  - 分享这一集
  - 下载到本地

#### 4.2.3 后台播放
应用支持后台播放功能：
- 切换到其他应用时继续播放
- 锁屏状态下继续播放
- 可在系统媒体控制中心控制播放

#### 4.2.4 播放进度保存
- 播放进度自动保存
- 下次打开该单集时从上次位置继续
- 支持跨设备同步（通过云端账号）

### 4.3 下载功能

#### 4.3.1 下载单集
1. 在单集列表或详情页点击下载按钮
2. 单集会添加到下载队列
3. 在"下载"标签页查看下载进度

#### 4.3.2 下载管理
**正在下载的单集：**
- 显示下载进度百分比
- 显示进度条
- 可暂停/继续下载
- 可取消下载

**已下载的单集：**
- 显示文件大小
- 可直接播放
- 可添加到待听队列
- 可删除下载文件

#### 4.3.3 批量管理
1. 在"下载"标签页点击"管理"按钮
2. 进入编辑模式，选择多个下载项
3. 可批量添加到待听队列
4. 可批量删除下载文件

#### 4.3.4 离线收听
- 已下载的单集会优先使用本地文件播放
- 无网络时仍可播放已下载的内容
- 下载文件存储在应用私有目录

### 4.4 队列管理

#### 4.4.1 添加到队列
有以下几种方式添加单集到队列：
1. 在单集列表点击"+ 待听"按钮
2. 在播放页面点击"添加到队列"
3. 批量从下载列表添加

#### 4.4.2 队列播放
- 队列按添加顺序排列
- 可以点击任意单集立即播放
- 播放完成后自动播放队列下一集（需在设置中开启）

#### 4.4.3 队列管理
- 查看队列中的单集数量
- 单独移除某个单集
- 清空整个队列
- 支持拖动排序（未来版本）

#### 4.4.4 自动续播
在设置中可开启自动续播功能：
- 当前单集播放完成后自动播放队列中的下一集
- 队列为空时停止播放

### 4.5 搜索功能

#### 4.5.1 搜索播客
1. 在搜索页面输入关键词
2. 搜索播客名称、作者或描述
3. 支持 RSS 地址直接订阅

#### 4.5.2 搜索单集
1. 在"单集"标签页顶部搜索框输入关键词
2. 搜索单集标题或简介内容
3. 实时显示搜索结果数量
4. 支持分页加载更多结果

#### 4.5.3 搜索结果
- 显示匹配的数量统计
- 高亮显示匹配关键词（未来版本）
- 支持点击查看详情

### 4.6 单集详情页

单集详情页提供以下信息和功能：

**单集信息：**
- 封面图片
- 单集标题
- 发布日期
- 时长信息
- 完整简介（支持展开/收起）
- 所属播客信息

**操作功能：**
- 播放按钮
- 添加到队列
- 下载到本地
- 分享单集
- 查看所属播客

### 4.7 播客详情页

播客详情页包含：

**播客信息：**
- 封面图片
- 播客名称
- 作者信息
- 分类标签
- 语言信息
- 完整描述
- 订阅日期
- 单集总数

**单集列表：**
- 显示该播客的所有单集
- 按发布日期倒序排列
- 显示播放状态
- 显示下载状态

**操作功能：**
- 订阅/取消订阅
- 刷新播客信息
- 查看官网链接
- 分享播客

## 六、常见问题

### 6.1 播放问题

**Q1：为什么设置了2倍速，返回主页后又变回1倍速？**  
A1：此问题已在 1.0.3 版本修复。现在播放速度会在迷你播放器和完整播放页面之间保持一致。

**Q2：后台播放时为什么会自动暂停？**  
A2：请确保已授予"后台运行权限"，并在系统设置中允许应用后台运行。

**Q3：播放进度为什么不保存？**  
A3：播放进度每秒自动保存。如果发现未保存，请检查应用是否有足够的存储权限。

### 6.2 下载问题

**Q4：为什么下载很慢或失败？**  
A4：可能原因：
- 网络连接不稳定
- 播客服务器响应慢
- 存储空间不足
建议：
- 切换到 WiFi 网络
- 检查可用存储空间
- 稍后重试

**Q5：下载的文件存放在哪里？**  
A5：下载文件存放在应用私有目录的 podcasts 文件夹中，其他应用无法访问。

### 6.3 订阅问题

**Q6：为什么有些播客无法订阅？**  
A6：可能原因：
- RSS 地址无效或已失效
- 播客源不支持的格式
- 网络连接问题
建议：
- 检查 RSS 地址是否正确
- 在浏览器中验证 RSS 地址可访问
- 尝试使用其他网络环境

**Q7：播客单集不更新怎么办？**  
A7：
- 新订阅的播客会在1分钟内频繁刷新
- 已订阅的播客每2小时自动刷新
- 可以手动下拉刷新播客详情页

### 6.4 其他问题

**Q8：如何导出/导入订阅？**  
A8：此功能将在未来版本提供，敬请期待。

**Q9：支持哪些音频格式？**  
A9：支持 HarmonyOS 系统支持的所有音频格式，包括 MP3、AAC、M4A、FLAC 等。

**Q10：如何反馈问题或建议？**  
A10：可通过以下方式联系：
- 应用内"关于"页面的反馈入口
- 发送邮件至开发者邮箱
- 在应用商店留言

### 7.1 高效管理订阅
1. 定期清理不再收听的播客

1. 使用1.5x-2x速度收听新闻类播客
2. 使用0.75x速度学习外语播客
3. 设置睡眠定时器，睡前收听

### 7.4 内容发现

## 十、隐私与安全

### 10.1 数据收集
应用仅收集必要的功能数据：
- 订阅的播客信息
- 播放历史记录
- 下载记录
- 应用设置

### 10.2 数据存储
- 所有数据存储在本地设备
- 不会上传到服务器
- 卸载应用后数据自动清除

### 10.3 权限使用
- 网络权限：仅用于获取播客内容
- 后台权限：仅用于后台播放
- 不会访问其他敏感权限

### 10.4 第三方服务
应用不使用任何第三方数据分析服务。
